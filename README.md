# Spike
Offline Web-app Valorant Spike Timer

(6s) Red is where spike is no longer defusable.

Interval (s)
Black : `45` to 31
White : `30` to 21
Yellow: `20` to 11
Orange: `10` to 7
Red   : `6` to 0

# Usage
Use the [web-app](https://xetsue.github.io/spike) directly.

Or, Download `index.html` and open in any browser. This web app do not require any internet connection to work.
